This is my website devoted to open source programming. Whenever possible I program in C++. If you would like to know why I will be including a number of articles under Why C++

antlr4

ANTLR (ANother Tool for Language Recognition) is a powerful parser generator for reading, processing, executing, or translating structured text or binary files. This is a fork of antlr/antlr4, the only thing which might be of interst is the c++17-support branch which I have modified the antlr C++ runtime to work better in C++17 by making antlrcpp::Any into an alias of std::any.

EzNetwork

A C++ library with the goal of easing the task of managing network connections. Since the C++ standard library lacks nework support. Hoping C++20 will provide so I only do work required to support my on projects.

PiDP-8-sim

A PDP-8 simulator for the Raspberry Pi aimed at running efficiently on the A+ model and working with the PiDP8 kit. Work on this project is suspended in favour of PiDP-8-sim-cpp

PiDP-8-sim-cpp

A complete re-write of my PDP-8 simulator project for the Raspberry Pi; still aimed at running efficiently on the A+ model and working with the PiDP8 kit. Currently porting to C++17. This work will feature in my Why C++ articles.

BITX40 Sketch

BITX40 sketch for Raduino. Forked from amunters/bitx40 for the odd time that I make changes. So far all my pull requests have been accepted so you should probably just go to the upstream repository.

ESP 8266

Arduino sketches for ESP 8266 based devices. a sketch which measures and reposrt temperature and humidity to a REST service. The sketch uses the ESP 8266 deep sleep mode between readings to reduce current consumption.

ParameterLib

An include only, template based C++ wrapper around GNU getopt_long.
