This method does the bulk of the work to complete realization of a socket.

Parameters
bind_connecteither ::bind() for a server or ::connect() for a client.
ai_family_preferencethe preferred address family AF_INET, AF_INET6 or AF_UNSPEC

Either bind or connect the socket. On error collect the message, close the socket and set it to error condition. Try the next connection or return error.

Store the selected peer address

Allow socket reuse.

Finds a connection specification that allows a socket to be created and bound preferring the provided address family preference, if any. If the socket fd is successfully created this method also calls:
