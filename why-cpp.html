Why C++

I have been programming in C++ since the beginning in the 1990s when the Cfront compiler was first released. The history of C++ has not really been smooth. The first implementations of generic programming with Cfront were particularly gruesome. I have programmed in other languages that are as old, or even older, that are still around. But none of those languages have improved and advanced as much, or as quickly as C++. There are other languages, Rust in particular, that seek to include some of the best features of C++. I mention Rust specifically because it is intriguing. Unfortunately it does not have a standards body, which I consider crucial for any code that I want to be supported into the future.

In the articles I will write under this directory I will detail how I use some of the modern features of C++. Maybe someone will find these useful or interesting.
